Blog admin login app

Project:

This project was a basic administrative user interface (GUI) built using Java Swing. The app can read/delete usersâ€™ info from the server database. The app is communicated with the server database by REST API.

Project Difficulties:

The biggest challenge was to implement appropriate design patterns to make the app more readable and maintainable. Another one was to figure out how to send HTTP requests and receive responses in Java.

The solution:

I implemented MVC, Adapter and Observer design patterns. The adapter pattern could convert HTTP responses to JTable content, and the observer pattern could update changes from the database to the JTable. I used the HTTPClient package to handle the HTTP request and responses.
