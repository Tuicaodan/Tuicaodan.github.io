Fungi research data analysis and display website

Project:

This is the start point where I found my passion for coding. The project came from the idea of automated data analysis of raw experiment data from the luminescence plate reader using Python. The raw data were converted to tidy data format. The tidy data were then analysed and displayed by using Pandas, Plotly and Dash. This project proved my capability to acquire new skills quickly and showed my logic of programming even with a limited coding experience.

Notable Features:
  • Automated data analysis of raw experimental data from luminescence plate reader using Python
  • Interactive graphical interface for better data analysis
  • Web app to display microbiology research data (Dash, Plotly)
