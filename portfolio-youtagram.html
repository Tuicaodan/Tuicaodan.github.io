Youtagram

Project:

This is a web app that mock Instagram. Instead of post photos, login users can post Youtube video links. Users can also read and comment on posts. The author of the post is allowed to update or delete the post.

The aim of this project is to learn how to implement React, TypeSript, Apollo Client for GraphQL, Styled Components, TailwindCSS, Github OAuth and React Context API together.

Reflections:
  • There were lots of skills need to learn during this project. The learning curves are all pretty sharp. So I'm really proud of myself that I can put these together and finish this project within a month and a half.
  • TypeScript is a great match with React. I'm glad that I chose to use TypeScript over JavaScript when starting the project. Because of the compiling, I was able to avoid lots of potential errors in props passing, Context API passing, etc. Otherwise, I can imagine the troubleshooting will be a great nightmare for React rookies.
  • styled-components and Tailwind CSS is also a great combination. This combination makes writing components much easier. The styled-components also make some of the components reusable. However, even doing this small project, sometimes I felt it was hard to find the correct components. I think I will need some components library like Storybook. Components testing is another thing I should do.
  • The Github OAuth feature was implemented in the project. This is a good experience to learn how to integrate OAuth into a react project.
  • I also used React Content APIs as the state management. With this global state management tool, props drilling and lifting can be avoided. This is a much easier way to manage auth content, theme content.
